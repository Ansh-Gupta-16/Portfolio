PROJECT DESCRIPTION:
This project is a single-page static portfolio website developed using HTML, CSS, and JavaScript to showcase the profile, skills, projects, and experience of a software developer. 
The HTML file defines the structure of the website, including sections such as Home, About, Skills, Projects, Experience, and Contact. 
It also includes navigation links that allow users to smoothly scroll between different sections of the page.
The CSS file is responsible for the visual design and layout of the website. 
It implements a modern and responsive interface with cards, grids, and styling for different sections. 
It also supports both dark and light themes using CSS variables to improve the user experience.
The JavaScript file adds interactivity to the website. 
It enables smooth scrolling navigation, a mobile menu toggle for smaller screens, and a dark/light mode switch that stores the user’s preference using local storage. 
It also dynamically updates the footer year and adds links to external profiles such as GitHub and LinkedIn.
Overall, the project demonstrates frontend development skills by combining structured layout, responsive design, and interactive features in a clean and user-friendly portfolio website.
